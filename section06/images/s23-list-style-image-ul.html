목록 스타일 - 순서 있는 목록 스타일 변경

순서있는 목록의 불릿 대신에 이미지를 넣을 수있다.

list-style-image : none | url("...")

  • none : 기본값, list-style-type에서 지정한 속성으로 표시
  • 이미지 : url("image.jpg")

list-style-image : image

  • 이력서
