Text Align은 텍스트를 정렬한다

text-align: start | end | left | right | ...

  • start :
  • end :
  • left :
  • right :
  • center :
  • justify :
  • match-parent :

start: 현재 텍스트 줄의 시작 위치에 맞춰서 문단을 정렬

end: 현재 텍스트 줄의 끝위치에 맞추어

left: 왼쪽에 맞추어

right: 오른쪽에 맞추어

center: 가운데에 맞추어

justify: 양쪽에 맞추어 문단을 정렬, 양쪽 맞추기를 위해 텍스트의 내용을 많이 추가했습니다

match-parent: 부모 요소를 따라 문단을 정렬
